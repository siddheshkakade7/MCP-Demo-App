# mcp-demo-app Documentation

Minimal Flask demo project structure and solution.

Structure:

mcp-demo-app/
├── app/
│   ├── __init__.py
│   ├── routes.py
│   ├── mcp_connector.py
├── static/
│   ├── style.css
│   └── script.js
├── templates/
│   └── index.html
├── requirements.txt
├── run.py
└── README.md

To run locally (assuming you have a Python virtualenv):

1. Install deps: pip install -r requirements.txt
2. Start: python run.py
